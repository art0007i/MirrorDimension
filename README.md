# MirrorDimension

Mod for [ChilloutVR](https://chilloutvr.net/) which allows you to mirror flip your player relative to others. This means that, for example, if you raise your left hand, others will see it as the right hand. As a side effect, the entire world will look mirrored from your perspective.

## Building

To build the mod you need to set up an environment variable `CVRPATH` with the value set to the location of your ChilloutVR install (for example: `C:\Program Files (x86)\Steam\steamapps\common\ChilloutVR`)

Afterwards simply run `dotnet build`.
