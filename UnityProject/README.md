# MirrorDimension Unity Project

Unity project which generates a asset bundle required for the MirrorDimension mod to work. The generated package is placed inside `../Resources`.

## Usage

1. Open the unity project (I made it in 2021.3.6f1, but it should open in any version)
2. Open the main scene (Assets/Main.unity)
3. Click the "Bundler" GameObject in the hierarchy
4. If you want the files to be automatically copied to the resources folder, make sure the "Copy to Resources Dir" checkbox is checked
5. Click "Build Bundle" inside the inspector window
